# SDL3 Platformer Project

This is my SDL3-based platformer game built with C++, SDL3, and TinyXML2.

-----------------------------------------------------------------------------

## Requirements

- CMake
- C++ Compiler (Visual Studio / g++ / clang)
- vcpkg

------------------------------------------------------------------------------

## Dependencies

Handled automatically by vcpkg:

- SDL3
- SDL3_image
- tinyxml2
